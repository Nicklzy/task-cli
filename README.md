# to-do-list
> A command line tool that allows you to create a To-Do lists.
## Documentation
```shell
npm install -g @nicklzy/to-do-list
```
## Usage
```shell
t
```
```shell
> exit
  [_] buy food
  [x] make food
  + add a task

```
```shell
? Which task do you want to choose? [_] buy food
? What do you want to do? (Use arrow keys)
> exit
  set as undone
  set as complete
  update title
  delete
```
### clear list
```shell
t clear
```

### quickly add to-do
```shell
t add taskName
```
